# 🤖 Bilibili AI 总结机器人 (Bilibili AI Summarizer)

这是一个基于 Python 和大语言模型（LLM）的 B站全自动总结机器人。当你在 B站视频评论区 `@本机器人` 时，它会在后台自动抓取视频字幕，进行深度思考与总结，将**万字完整研报保存到本地**，并将**精华要点直接在评论区回复**给你！
（本项目仅为大一小白的试手项目，仅为个人学习使用，个人知识储备有限，有很多不会的东西，欢迎大家多多交流）

## ✨ 核心功能 (Features)

* **🎧 实时监听：** 24小时异步轮询监听 B站未读的 `@` 消息。
* **📜 智能抓取：** 自动解析视频 BV 号，精准提取底层 `cid` 并下载官方 CC/自动字幕。
* **🧠 深度解析：** 接入大语言模型（默认支持火山引擎/DeepSeek），基于专业提示词生成高质量的“结构化深度研报”。
* **💾 本地知识库：** 自动绕过 B站字数限制，将完整的 Markdown 格式深度研报保存到你的电脑桌面。
* **💬 评论区自动回复：** 提取“一句话总结”和“核心要点”，在视频评论区自动 `@召唤者` 并回复精华内容。
* **🛡️ 高容错设计：** 内置强大的正则清洗模块，完美处理 AI 幻觉、多余空格和非法特殊字符，确保极高的回复成功率。

## 🚀 快速开始 (Getting Started)

### 1. 安装依赖环境

在你的终端（命令行）中运行以下命令安装必需的第三方库：

```bash
pip install bilibili-api-python openai httpx
```

### 2. 获取并配置 B站凭证 (Cookies)

本项目需要一个 B站账号作为“机器人”运行。为了账号安全，我们使用 Cookie 登录：
1. 在浏览器登录你的机器人账号。
2. 按 `F12` 打开开发者工具，进入 `Application` (应用) -> `Cookies`。
3. 找到并复制以下三个字段的值：`SESSDATA`, `bili_jct`, `DedeUserID`。
4. 将它们填入代码头部的配置区（注意：千万不要将含有真实 Cookie 的代码上传到公开仓库！）。

### 3. 配置 AI 模型 API (以火山引擎为例)

本项目默认使用兼容 OpenAI 格式的 API 接口。
1. 注册获取你的 `API_KEY`。
2. 如果使用火山引擎（Volcengine），请确保模型名称填写为对应的接入点 ID（如 `ep-xxxxxx`）。

### 4. 运行机器人

在配置好所有参数后，直接运行 Python 脚本：

```bash
python bilibili_ai_bot.py
```

当控制台打印出 `✅ 开始巡逻...` 时，你的机器人就已经正式上岗了！快去 B站评论区 `@它` 试试吧！

* **账号安全：** 请**绝对不要**将填有真实 Cookie 和 API Key 的代码上传到公开的网络仓库中！上传前请务必脱敏。
* **使用规范：** 请遵守 Bilibili 社区规范。建议适当限制轮询频率（代码中默认为 10 秒），避免频繁请求导致账号被风控或封禁。本项目仅供个人学习交流使用。
